# 💬 Let's Connect — CometChat Integration (React + TypeScript)

This project integrates the **CometChat UI Kit** into a React + TypeScript application to enable **one-to-one chat** functionality between users.

## 🚀 Features Implemented

- ✅ User-to-User Chat (1:1)
- ✅ Message Sending & Receiving
- ✅ Real-time Messaging via CometChat
- ✅ Login using UID-based Authentication
- ✅ Logout Functionality
- ✅ App Credential Management (App ID, Region, Auth Key)
- ✅ Dark/Light Mode Toggle for Credentials Page
- ✅ Responsive UI using TailwindCSS & Lucide Icons
- ✅ Persistent Credential Storage via `localStorage`
- ✅ User Presence Indicator
- ✅ UIKit Initialization with Custom Settings

> ❌ Group Chat features were **not applied** in this project.

---

## 📦 Technologies Used

- React (with Vite)
- TypeScript
- Tailwind CSS
- Lucide React Icons
- React Router DOM
- CometChat UI Kit (React)

---

## 🧠 Learnings & Understanding

While integrating CometChat, I learned how to:

- Configure and initialize the CometChat UI Kit using `UIKitSettingsBuilder`
- Handle user authentication via UID, App ID, Region, and Auth Key
- Use CometChat’s built-in components for real-time messaging
- Customize UI with dark/light theme compatibility
- Store and reuse credentials locally
- Manage navigation and protected routes
- Use the CometChat SDK efficiently without implementing unnecessary features like groups
